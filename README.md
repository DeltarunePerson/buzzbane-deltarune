# buzzbane-deltarune
All current acapellas by Buzzbane.

# How to use:
1. Download the ZIP folder by clicking Code, then Download as ZIP
2. Unzip the folder.
3. Open the acapellas folder.
4. Select all .ogg files and move them into the DELTARUNE music folder (The music folder is located in: "Program Files (x86)/Steam/steamapps/common/DELTARUNE/mus" on Windows.)
5. While moving, Windows will notify you that some files already exist. Click on "Replace the files in the destination"
