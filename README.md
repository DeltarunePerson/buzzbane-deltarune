# buzzbane-deltarune
All current acapellas by Buzzbane.

# How to use:
1. Download the newest release in Code -> Download ZIP
2. Unzip and open the folder. Go to the music folder.
3. Select all .ogg files and move them into the DELTARUNE music folder (The music folder is located in: "Program Files (x86)/Steam/steamapps/common/DELTARUNE/mus" on Windows. You can also find it in Steam -> DELTARUNE -> Manage -> Browse local files.)
4. While moving, Windows will notify you that some files already exist. Click on "Replace the files in the destination".

If you want to go back to the original songs, just delete and redownload the game (Your saves will not get wiped).
